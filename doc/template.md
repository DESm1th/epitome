volsmooth
---------
Usage: template <func_prefix> <opt1> <opt2> <opt3> <opt4>

+ func_prefix -- functional data prefix (eg., smooth in func_smooth).
+ opt1 -- a string.
+ opt2 -- a float.
+ opt3 -- an int.
+ opt4 -- a string.

This just prints to the screen. but is a useful template for constructing your own modules from.

Prerequisites: init_*.
